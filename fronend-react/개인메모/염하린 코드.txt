회원가입 페이지
소셜계정 가입
1. 소셜계정 가입 버튼을 누른다.
2. 소셜계정 가입 팝업이 뜬다.
3. 개인정보 입력후 가입완료 페이지로 넘어간다.

일반회원 가입

1. 아이디 중복확인 버튼을 누른다. 
 1-1 중복인 경우 다른 아이디 입력하라는 경고 메시지를 띄운다/ 중복이 아닌 경우 가입 가능한 아이디라는 메시지를 띄운다.
 1-2 중복 확인 후 중복확인완료로 바뀐다.
2. 패스워드가 다를시 패스워드가 다르다는 경고 메시지를 띄운다.
 2-1 눈모양 아이콘을 누르면 비밀번호가 공개된다.
3. 통신사 선택 카테고리 버튼을 누르면 통신사를 선택할 수 있는 목록이 나온다.
 3-1 본인인증 버튼을 누르면 본인인증 팝업이 뜬다.
 3-2 팝업창에서 본인 인증 후 인증버튼이 완료로 바뀐다.
4. 회원가입 버튼을 누르면 회원 가입 완료 페이지로 넘어간다.
 4-1 중복확인/본인인증 을 안했을 경우 완료하라는 경고 메시지를 띄운다.

회원가입 완료 페이지
시작 버튼을 누르면 로그인 된 메인페이지로 넘어간다.

회원정보 수정 페이지

1. 프로필 사진 등록 버튼을 누르면 이미지 파일을 첨부하는 창이 뜬다.
 1-1 사진 등록시 사람아이콘이 이미지 파일로 변경된다.
 1-2 사진이 등록되어 있을 경우 삭제버튼을 눌러 등록된 이미지를 삭제한다.
2. 변경할 이름를 입력하면 (개명과 같은 사유로 변경 가능하도록 만들기)
 2-1 중복확인 버튼을 눌러 중복 확인을 한다.
 2-2 중복인 경우 다른 아이디 입력하라는 경고 메시지를 띄운다/ 중복이 아닌 경우 가입 가능한 아이디라는 메시지를 띄운다.
 2-3 중복 확인 후 중복확인완료로 바뀐다.
3. 패스워드가 다를시 패스워드가 다르다는 경고 메시지를 띄운다.
 3-1 눈모양 아이콘을 누르면 비밀번호가 공개된다.
4. 닉네임을 입력 받을 수 있는 란에 입력받아 등록 수정 가능하도록 한다.
 4-1 중복확인 버튼을 눌러 중복 확인을 한다.
 4-2 중복인 경우 다른 닉네임을 입력하라는 경고 메시지를 띄운다/ 중복이 아닌 경우 사용 가능한 닉네임이라는 메시지를 띄운다.
 4-3 중복 확인 후 중복확인완료로 바뀐다.
 4-4 이미 등록 되어 있는 경우 수정 버튼을 눌러 4-1,4-2,4-3 과정을 거친다.
5. 수정완료 버튼을 누르면 변경된 정보가 뜨는 회원정보 조회 페이지로 넘어간다.
 5-1 본인인증/중복확인을 안했을 경우 완료하라는 경고 창이 뜬다.
 5-2 변경된 정보가 없을 경우 변경된 정보가 없습니다. 진행하시겠습니까? 라는 메세지를 띄운다.
