4. tailwind (프로젝트 메인 css)

설치순서

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

두개 다 해주셔야 설정폴더까지 생성됩니다

+ 추가 tailwind.config.js 설정

// tailwind.config.js

/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
    "node_modules/preline/dist/*.js",
  ],
  darkMode: 'class',
  theme: {
    screens: {
      'sm': '640px',
      'md': '768px',
      'lg': '1024px',
      'xl': '1280px',
      '2xl': '1536px',
    },
    extend: {},
  },
  plugins: [
     require('@tailwindcss/forms'),
     require('preline/plugin'),
  ],
}

그리고 index.css에 

@tailwind base;
@tailwind components;
@tailwind utilities;

추가

https://tailwindcss.com/docs/guides/vite <<<<< vite 전용 설치법 가이드
